{% admonition type="info" name="Attention" %}
Lending Protocol and Single Asset Vaults are disabled on Devnet. You have two options for testing these features:
1. Run `rippled` in [stand-alone mode](https://xrpl.org/docs/concepts/networks-and-servers/rippled-server-modes#stand-alone-mode) and enable `LendingProtocol` and `SingleAssetVault` in the [rippled.cfg file](https://xrpl.org/docs/infrastructure/testing-and-auditing/test-amendments).
2. Connect to the Lending Protocol-specific Devnet at `http://lend.devnet.rippletest.net:51234/`.
{% /admonition %}
