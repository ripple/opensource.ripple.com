# Programmability

A programmability layer, powered by a WebAssembly (WASM) engine, allows users to create custom functions to extend the core functionality of the ledger. Currently, the only extensible functionality suite is Smart Escrows, but further work is planned.

[XLS-102](https://xls.xrpl.org/xls/XLS-0102-wasm-vm.html) specifies the WebAssembly engine built into the core XRP Ledger server.
